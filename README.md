# .NET & C# Backend Development Journey

Welcome to my central learning and practice hub for .NET Core and C#. This repository serves as a practical sandbox and technical roadmap designed to master production-ready backend engineering, advanced language features, and clean software architecture.


---

##  Tech Stack & Tools
* **Language:** C# (Modern features)
* **Framework:** .NET Core / ASP.NET Core
* **IDE/Editor:** VS Code (on macOS Apple Silicon)
* **Database & Tools:** Entity Framework Core, LINQ, PostgreSQL / MS SQL

---

##  Core Focus Areas & Learning Roadmap
This repository is iteratively updated to cover a **13-week intensive technical roadmap**, moving from fundamental concepts to complex architectural patterns:

- [ ] **Advanced C# Mechanics:** Deep dive into `async/await`, tasks, delegates, events, and memory management.
- [ ] **Object-Oriented Programming (OOP) & SOLID:** Implementing highly decoupled, maintainable, and testable code structures.
- [ ] **Data Access Layer:** Mastering LINQ queries, advanced EF Core mappings, and repository patterns.
- [ ] **ASP.NET Core Web APIs:** Building robust RESTful services, custom middleware, global exception handling, and Fluent Validation.
- [ ] **Clean Architecture:** Designing scalable systems using layered/onion architecture, dependency injection, and CQRS principles.
